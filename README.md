# Global YouTube Creator Analytics – Power BI Dashboard

This project showcases an **interactive Power BI dashboard** built to analyze global YouTube creator performance and engagement metrics. The dashboard provides insights into video performance using key KPIs, visuals, and interactive filters.

## Dashboard Overview

The report contains a single analytics page highlighting engagement metrics and content performance.

### Key KPIs

* Total Comments
* Total Views
* Total Shares
* Total Likes

Each KPI includes **tooltips displaying the average values** for deeper analysis.

### Interactive Filters

Users can filter the dashboard using:

* Category
* Language
* Year
* Month

### Visualizations

* **Donut Chart:** Percentage of total videos by category
* **Pie Chart:** Distribution of total videos by language

### Data Tables

* **Video Details Table:** Video ID, Upload Date, Region, Duration, Likes, Comments, Shares, and Views
* **Content Analysis Table:** Most watched content by language

## Purpose

The dashboard helps analyze global YouTube content performance, engagement trends, and language-based audience preferences through interactive data exploration.

## Tools Used

* Power BI
* DAX Measures
* Interactive Filters & Visualizations
